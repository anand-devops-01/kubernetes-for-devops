# ☸️ Kubernetes for DevOps

A comprehensive Kubernetes learning repository designed for DevOps Engineers, Cloud Engineers, and Platform Engineers.

This repository covers Kubernetes concepts from beginner to advanced level with notes, YAML examples, interview questions, and practical cheat sheets.

---

## 📚 Topics Covered

- Kubernetes Introduction
- Kubernetes Architecture
- Pods
- ReplicaSets
- Deployments
- Services
- Namespaces
- Labels & Selectors
- ConfigMaps
- Secrets
- Volumes & Storage
- Networking
- Ingress
- Resource Limits
- Probes
- RBAC
- Jobs & CronJobs
- Troubleshooting
- Interview Questions
